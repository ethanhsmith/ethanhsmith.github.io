What is grid based design?

Grid based design is the use of rows and columns to present a webpage that uses a browsers dimensions to portray the websites data in a way that is molded to the browser in use. This type of web development is useful in reactive designs.

What is semantic structure?

Semantic structure was introduced in HTML5 to give meaning to plain old elements such as "div" which references where the element is being used. E.g footer, header, nav etc.

What is all the hype about responsive webpages?

Responsive webpages allow a website to be designed once and then to work for multiple devices. The website uses the browser to detect the size of the device being used and will react accordingly so that information is presented differently or that the layout changes significantly to match each device!
