How would you describe HTML, CSS and the DOM to your non-technical friend?

HTML is like the the building materials you have before you build a house, CSS is what decides how these materials are positioned and used to effectively build and style a structure and the DOM is the resulting floor plan (which can be used as a concept before doing CSS work).

What is boxifying design?

Boxifying design is the understanding that all elements on a webpage are positioned within a box, and to correctly format and position the contents of a webpage a series of boxes will be needed to allow proper formating, so the developer plans out how these boxes will be correctly utilised. For example: For a section such as a header to look how a developer wants, he may need first a header element, then an img element to hold a logo, another div element to hold a menu item, etc etc.

What is the box model?

The box model relates to every element, every element has a box which consists of margins, padding, borders and their internal content.
